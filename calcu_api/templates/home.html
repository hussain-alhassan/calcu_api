Welcome to Calcu

Calcu API is a calculator performing some operations using Web API and CLI (Command Line Interface). Also, it shows you reports about the times each operation has been used on daily, weekly, and monthly basis.


To use the Web API, click Calculate link the nav bar.
  • Daily Report shows the operations have been done in the last 24 hours.
  • Weekly Report shows the operations have been done in the last week.
  • Monthly Report shows the operations have been done in the last month.
